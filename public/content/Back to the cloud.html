Back to the cloud

So in the previous post I explained how I moved this site to be self hosted on a rasbperry pi. Well...it is now hosted back on the cloud, I'm using Google Cloud to host it.

I'm slowly moving towards something more akin to the Jamstack architecture. In this architecture, the idea is to remove the server component for the most part, at least the component serving up static files or hydrating views with data. This is replaced with javascript and API's.

All of the static assets are now pre-built if they have some data binded to them, and then pushed to the server. This eliminates the need for your server to this work.
